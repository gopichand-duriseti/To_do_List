# To-Do Application (Python CLI)

A simple command-line To-Do app using Python. Stores tasks in a CSV file.

## Features
- Add tasks
- Remove tasks
- View tasks
- Modify tasks
- Prevents duplicates
- Saves to tasks.csv

## How to Run
python to_do.py

## File Structure
to_do.py
tasks.csv
README.md

## Menu
1. Add Task
2. Remove Task
3. View Tasks
4. Modify Task
5. Exit
